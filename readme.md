# 🚀 Employee Hub – AI-Powered Workforce Management Platform

Employee Hub is a full-stack Employee Management System that streamlines HR operations by combining employee management, attendance tracking, performance evaluation, rewards management, analytics, and AI-powered predictions into a single platform.

The application is built using **React.js**, **Django REST Framework**, **PostgreSQL**, and **Machine Learning (Random Forest)** to help organizations automate HR processes and make data-driven decisions.

---

## ✨ Features

* Employee Management
* Attendance Tracking
* Performance Evaluation
* Rewards & Recognition
* HR Analytics Dashboard
* AI-Based Performance Prediction
* Promotion Readiness Prediction
* Attrition Risk Analysis
* Secure Authentication
* RESTful APIs

---

## 🧠 Machine Learning

The platform integrates a **Random Forest** model to generate intelligent HR insights, including:

* Employee Performance Score
* Promotion Readiness
* Attrition Risk
* Badge Recommendation
* Growth Track Suggestion

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Gajanan1904/AI-Powered-Employee-Management-System.git
cd AI-Powered-Employee-Management-System
```

### Backend Setup

```bash
cd EMS

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
